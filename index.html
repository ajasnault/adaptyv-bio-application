Personal project

Operating Room Capacity Simulator

A discrete-event simulation of a hospital operating-room block (rooms, recovery beds, staff) with a Swiss-franc cost module, used to arbitrate capacity trade-offs — e.g. is it better to add an operating room, or add recovery beds downstream? Illustrates a classic bottleneck: SSPI/recovery capacity can throttle throughput more than the operating rooms themselves.
